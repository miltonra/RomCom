__init__(path, data=None, **options)[source]§

Construct self from a .csv file or PD.DataFrame.

Parameters:

  • path (Store.Path) – The Path (file) to store self. A .csv extension is automatically appended.

  • data (DataFrame | None) – The data to store. If None, data is read from path, otherwise data is stored in path (which is overwritten if existing).

  • **options (Any) – Updates self.read_options if data is None, otherwise updates self.write_options.
