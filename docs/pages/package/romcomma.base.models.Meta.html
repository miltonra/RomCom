__init__(path, **data)[source]§

Construct self from a .json file or dict.

Parameters:

  • path (Path | str) – The Path (file) to store self. A .json extension is appended.

  • **data (Any) – The metadata to store. If absent, self.data is read from path, otherwise self.data=data is stored in path (which is overwritten if existing).
